## 常规项目目录参考

```bash
├─docs # 说明文档
│   └─STANDARD.md # eg. 规范文档
├─public
│  ├─favicon.ico
│  └─index.html
├─src
│  ├─assets # 静态资源
│  │  ├─css
│  │  │  ├─compatible.css # 兼容平台样式
│  │  │  ├─mixin.css # 样式类 eg.布局类、动画类
│  │  │  ├─variables.css # 变量 eg.主题色
│  │  │  └─index.css
│  │  ├─images
│  │  └─font
│  ├─components # 组件
│  ├─router # 路由
│  ├─services # 服务
│  │  ├─api
│  │  ├─ signature.ts # 加密
│  │  ├─ silentLogin.ts # 静默登陆
│  │  ├─ toLogin.ts
│  │  └─ httpInterceptors.ts # 配置、拦截器
│  ├─store # 状态管理
│  ├─utils # 工具
│  ├─view # 页面
│  ├─workers # 多线程
│  ├─nativeplugins # 原声插件
│  ├─platforms # 各平台专用页面
│  │  ├─app-plus
│  │  └─h5
```
