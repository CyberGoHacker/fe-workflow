# Prettier


## 安装
```bash
pnpm add prettier eslint-config-prettier eslint-plugin-prettier -D
```

## 解决与 eslint 冲突
```json
{
  "extends": [ 'prettier' ],
}
```

## .prettierrc 配置
```json
{
  "semi": false,
  "tabWidth": 2,
  "proseWrap": "never",
  "endOfLine": "auto",
  "printWidth": 100,
  "singleQuote": true,
  "trailingComma": "none",
  "vueIndentScriptAndStyle": true,
  "htmlWhitespaceSensitivity": "strict"
}
```
