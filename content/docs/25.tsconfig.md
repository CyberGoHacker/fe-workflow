# TSConfig

## 安装

```bash
pnpm add typescript @typescript-eslint/parser @typescript-eslint/eslint-plugin -D
```

## 配置 .eslintrc

```json
{
  "plugins": ["@typescript-eslint"],
  "parserOptions": {
    "parser": "@typescript-eslint/parser"
  }
}
```
